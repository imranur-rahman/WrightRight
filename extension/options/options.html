LanguageTool Configuration
API endpoint for LanguageTool checks. Use the public API or your self-hosted instance.
Self-hosting LanguageTool: LanguageTool is open-source (LGPL). Run it locally with:
java -jar LanguageTool-*.jar --http --port 8081
Then set the endpoint to http://localhost:8081/v2/check. No API key required for self-hosted instances.
